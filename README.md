# 🎵 VK Music Skipper

Это расширение для Chrome, которое автоматически переключает песню в ВК, как только она проиграла определенное количество секунд. Идеально подходит, чтобы быстро прослушивать новинки или плейлисты.

## 🛠 Как установить (за 1 минуту):

1. **Скачай и распакуй** этот архив в отдельную папку.
2. Открой браузер (Chrome, Яндекс, Edge) и перейди по ссылке: `chrome://extensions/`
3. В правом верхнем углу включи тумблер **"Режим разработчика"** (Developer mode).
4. Нажми кнопку **"Загрузить распакованное расширение"** (Load unpacked).
5. Выбери папку, в которой лежат файлы расширения.
6. **Готово!** Теперь зайди в ВК и обнови страницу.

## ⚙️ Как пользоваться:

- Нажми на иконку «пазла» в панели браузера и закрепи **VK Music Skipper**.
- Кликни по иконке расширения, введи нужное количество секунд (например, 30 или 60) и нажми **«Сохранить»**.
- Наслаждайся музыкой!
